✅ Create a question
- Write question description in the text area.
- Write answer options.
- If you need more input options, hit 'Add option'
- If you want to remove input options, hit 'Remove option'
- Options' minimum is 2 and maximum is 4.
- Choose correct answer by checking radio button at the left of each option

✅ Update a question
- Hit 'Move Data to the form to update' button on the question to update
- 'Save to DB' button will be disabled
- If you want go back to create a question, hit 'Refresh' button
